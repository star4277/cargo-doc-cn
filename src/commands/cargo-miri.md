# cargo-miri(1)

## 名称

cargo-miri --- 在 Miri 中运行二进制 crate 与测试

## 描述

这是一个外部命令，作为可选组件随 Rust 工具链发布。
它不是 Cargo 的内置命令，可能需要额外安装。

该命令仅在 [nightly](https://doc.rust-lang.org/book/appendix-07-nightly-rust.html) 通道可用。

关于使用和安装信息，
请参见 <https://github.com/rust-lang/miri>。
