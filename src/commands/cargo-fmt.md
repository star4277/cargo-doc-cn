# cargo-fmt(1)

## 名称

cargo-fmt --- 使用 rustfmt 格式化当前 crate 的所有 bin 与 lib 文件

## 描述

这是一个外部命令，作为可选组件随 Rust 工具链发布。
它不是 Cargo 的内置命令，可能需要额外安装。

关于使用和安装信息，
请参见 <https://github.com/rust-lang/rustfmt>。
