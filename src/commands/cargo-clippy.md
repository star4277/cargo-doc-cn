# cargo-clippy(1)

## 名称

cargo-clippy --- 检查 package，以捕获常见错误并改进 Rust 代码

## 描述

这是一个外部命令，作为可选组件随 Rust 工具链发布。
它不是 Cargo 的内置命令，可能需要额外安装。

关于使用和安装信息，
请参见 [Clippy Documentation](../../clippy/index.html)。
